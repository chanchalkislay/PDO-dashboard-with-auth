# PDO Dashboard — Demo (June 2026)

Pune Divisional Office market-share dashboard. Data through May 2026.

## Run locally

**macOS:** Double-click `run_demo.command`  
**Windows:** Double-click `run_demo.bat`  
Launches at http://localhost:8502

## Streamlit Cloud deployment

- Main file: `app/app.py`
- Branch: `demo-jun26`

## Data coverage

- All 6 OMCs · Pune DO territory (Pune, Ahmednagar, Satara districts)
- Historical: FY 2019-20 through FY 2025-26
- Current: April 2026 and May 2026
- 16 analytics tabs
